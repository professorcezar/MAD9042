# Assignment A1

# Questions (pick ONLY ONE)

  • We can build amazing games using AR. What else could we create, build, invent, code that could help people with their daily lives?

hint: Think about a lifestyle, profession, affilitation or mission, and use this context to guide you on your answer.

  • How could you integrate AR in your current project/job?

hint: Think small. Which little feature of you app, or process in your job could be better done if you could use AR on it?

  • What are people really using AR for?

hint: Is AR just a gimmick or is it really being used in production apps? Who is using it? What are the benefits?

How to answer and submit this assignment

  • You will create and submit a file called last-first-a1.txt (e.g cezar-vladimir-a1.txt)
  • Use plain txt files. No markups. No word docs. No pdf. No arcane formats.
  • The file must contain the question you are answering.
  • The file must contain the answer to the question. (Very important!)
  • The total number of words in the app should be between 256 and 512.
  • When you're done editing, open Brightspace and go to the Activities > Assignments page.
  • Go to the A1 assignment. Upload your file.

Important

  • I will be looking for ideas, reasoning, and context.
  • Get to the point ASAP: aim for the 300 word mark.
